## Security Is Layered

No single control is sufficient. A secure system applies defense in depth — multiple independent layers so that a breach of one does not compromise everything:

```
User Input → Input Validation → Business Logic → Data Layer
     ↓              ↓                  ↓              ↓
Rate Limiting   Sanitization      AuthZ Checks    Encryption
WAF/CDN         Parameterization  Least Privilege  Audit Log
```
