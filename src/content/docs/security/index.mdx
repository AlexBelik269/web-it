## Security Is Layered

No single control is sufficient. A secure system applies **defense in depth** — multiple independent layers so that a failure in one does not compromise everything:

```
Network Layer     → Firewall, IDS/IPS, DDoS protection
Application Layer → Input validation, WAF, rate limiting, CSP headers
Identity Layer    → Authentication, MFA, least-privilege authorization
Data Layer        → Encryption at rest, parameterized queries, secrets management
Detection Layer   → Logging, monitoring, alerting, EDR
Response Layer    → Incident playbooks, forensics, backup and recovery
```
