## How an LLM Answers a Question

```mermaid
sequenceDiagram
    participant User
    participant Tokenizer
    participant Transformer
    participant Decoder
    participant User as User

    User->>Tokenizer: "What is the capital of France?"
    Tokenizer->>Transformer: [Token IDs: 1023, 318, 262, ...]
    Transformer->>Transformer: Attention layers process context
    Transformer->>Decoder: Probability distribution over next token
    Decoder->>Decoder: Sample "Paris" (highest probability)
    Decoder->>User: "The capital of France is Paris."
```
