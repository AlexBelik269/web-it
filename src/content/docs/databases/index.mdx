    Q["SQL Query"] --> P["Parser\n(syntax check)"]

## Relational vs NoSQL at a Glance

```mermaid
flowchart TD
    Q{"What's your\nprimary need?"}
    Q -->|"Complex queries, joins,\nstrong consistency"| REL["Relational DB\nPostgreSQL / MySQL"]
    Q -->|"Flexible schema,\nhorizontal scale"| NOSQL["NoSQL"]
    NOSQL -->|"JSON documents"| DOC["Document\nMongoDB"]
    NOSQL -->|"Cache, sessions,\nhigh throughput"| KV["Key-Value\nRedis"]
    NOSQL -->|"Relationships,\ngraph traversal"| GRAPH["Graph\nNeo4j"]
    NOSQL -->|"Time-series,\nanalytics"| COL["Columnar\nCassandra"]
```
