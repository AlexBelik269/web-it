A comprehensive reference of protocols organised by function. Use this as a quick lookup when configuring firewalls, reading packet captures, or hardening network services.

## Application Protocols — Remote Access

| Protocol | Port(s) | Transport | OSI | Notes |
|---|---|---|---|---|
| **SSH** | 22 | TCP | 7 | Encrypted remote shell, file transfer (SFTP/SCP), tunnelling |
| **Telnet** | 23 | TCP | 7 | Plaintext remote shell — **insecure, avoid** |
| **RDP** | 3389 | TCP (UDP opt.) | 7 | Windows Remote Desktop Protocol |
| **VNC** | 5900 | TCP | 7 | Virtual Network Computing — cross-platform GUI |
| **X11** | 6000–6063 | TCP | 7 | X Window System remote display |
| **TeamViewer** | 5938 | TCP/UDP | 7 | Proprietary remote access |
